# Huzaifa Alahmad Portfolio Frontend

React + TypeScript + Vite portfolio frontend designed for hiring managers, recruiters, tech leads, founders, and clients.

## Run

```bash
npm install
npm run dev
```

Backend expected at:

```bash
http://localhost:5000/api/v1
```

Create `.env` from `.env.example` if needed.

## Admin test

Open `/admin/login` and use your backend admin credentials.

## Fonts

The design is prepared for your custom brand fonts. Place your licensed font files locally in `public/assets/fonts/` and update `src/styles/fonts.css` if you want the exact uploaded fonts applied.
