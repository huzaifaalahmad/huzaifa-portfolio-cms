# Huzaifa Portfolio CMS

A modern full-stack portfolio and personal CMS built for an Information Engineer, AI Prompt Engineer, Full-Stack Developer, and Visual Designer.

This project is not just a static portfolio. It includes a real backend, authentication, admin dashboard, dynamic content management, contact message handling, and a highly animated responsive frontend.

## Features

- Modern animated portfolio UI
- Arabic / English language support
- Dark / light mode
- Admin login system
- Protected admin dashboard
- Dynamic projects management
- Dynamic skills management
- Dynamic experience management
- Dynamic education and certifications management
- Contact form connected to backend
- Contact messages dashboard
- JWT authentication
- Refresh token support
- CSRF protection
- Rate limiting
- MySQL database with Prisma ORM
- Responsive design
- Production build ready
